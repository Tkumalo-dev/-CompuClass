## Getting Started

### First Time Setup

1. **Download the App**
   - Install Expo Go from App Store (iOS) or Google Play (Android)
   - Scan the QR code provided by your institution

2. **Create Your Account**
   - Tap "Sign Up" on the welcome screen
   - Enter your full name
   - Enter your email address
   - Create a secure password
   - Tap "Sign Up"

3. **Login**
   - Enter your email and password
   - Tap "Login"

### Account Types
- **Student Account**: Default for all users
- **Lecturer Account**: Use email `lecturer@compuclass.com`

## For Students

### Dashboard (Home Screen)

Your dashboard shows:
- Quick access cards to main features
- Recent activities
- Assigned quizzes
- Learning progress

**Navigation:**
- Tap the **Home** icon at the bottom to return to dashboard
- Swipe from left edge to open the side menu

### Learning Materials

**To Access Materials:**
1. Tap the menu icon (☰) at the top right
2. Select "Learning Materials"
3. Browse available documents and resources
4. Tap any document to view or download

**Supported Formats:**
- PDF documents
- PowerPoint presentations
- Images and diagrams

### Taking Quizzes

**To Start a Quiz:**
1. From Dashboard, tap on an assigned quiz
2. Read the quiz description
3. Tap "Start Quiz"
4. Answer each question by selecting an option
5. Tap "Next" to move to the next question
6. Tap "Submit" when finished

**Quiz Features:**
- Progress indicator shows questions completed
- Review your answers before submitting
- See your score immediately after submission
- View correct answers in the review section

### PC Lab (3D Learning)

**To Explore PC Components:**
1. Tap menu icon (☰)
2. Select "PC Lab"
3. View the 3D computer model
4. **Drag** to rotate the model
5. Tap "Expand" for fullscreen view
6. Tap component cards to view individual parts:
   - Motherboard
   - CPU (Processor)
   - RAM (Memory)
   - GPU (Graphics Card)
   - Storage (Hard Drive/SSD)
   - PSU (Power Supply)

**3D Controls:**
- **Drag**: Rotate the model
- **Pinch**: Zoom in/out (if supported)
- **Tap**: Select component

### Windows 11 Simulator

**To Practice Windows:**
1. Open side menu
2. Select "Windows 11"
3. Explore the simulated Windows interface
4. Practice common tasks safely

### Troubleshooting Guide

**To Access Help:**
1. Open side menu
2. Select "Troubleshooting"
3. Browse common PC problems and solutions
4. Follow step-by-step guides

### Profile & Settings

**To Update Your Profile:**
1. Tap the **Profile** icon at the bottom
2. View your information
3. Tap "Edit Profile" to make changes
4. Tap "Logout" to sign out

**To Change Settings:**
1. Open side menu
2. Select "Settings"
3. Toggle Dark Mode on/off
4. Adjust other preferences

## For Lecturers

### Lecturer Dashboard

Your dashboard provides:
- Quick action buttons
- Folder management
- Student progress overview
- Class management tools

### Creating Folders

**To Organize Content:**
1. Tap "Create New Folder"
2. Enter folder name
3. Add description (optional)
4. Tap "Create"

**To Delete a Folder:**
1. Find the folder in your list
2. Tap the trash icon
3. Confirm deletion

### Uploading Content

**To Upload Documents:**
1. Tap "Upload Content" button
2. Select a folder (or create new one)
3. Tap "Choose File"
4. Select document from your device
5. Enter document title
6. Tap "Upload"

**Supported File Types:**
- PDF (.pdf)
- PowerPoint (.ppt, .pptx)
- Images (.jpg, .png)

### Creating Quizzes

**Manual Quiz Creation:**
1. Tap "Create Quiz"
2. Select a folder
3. Enter quiz title and description
4. Set passing score (default: 70%)
5. Add questions:
   - Enter question text
   - Add 4 answer options
   - Mark the correct answer
6. Tap "Save Quiz"

**AI-Powered Quiz Creation:**
1. Tap "Create Quiz"
2. Select "Generate with AI"
3. Upload a PDF document
4. Choose number of questions (5-20)
5. Wait for AI to generate questions
6. Review and edit questions
7. Tap "Save Quiz"

### Managing Classes

**To Create a Class:**
1. Tap "Class Management"
2. Tap "Create New Class"
3. Enter class name and description
4. Tap "Create"

**To Add Students:**
1. Open a class
2. Tap "Add Students"
3. Select students from the list
4. Tap "Add Selected"

**To Assign Quizzes:**
1. Open a class
2. Tap "Assign Quiz"
3. Select quiz from your folders
4. Tap "Assign to Class"

### Viewing Student Progress

**To Track Performance:**
1. Tap "Student Progress"
2. View list of all students
3. Tap a student to see:
   - Quiz attempts and scores
   - Materials viewed
   - Overall progress
4. Filter by class or date

### Managing Quiz Details

**To View Quiz Results:**
1. Navigate to a folder
2. Tap on a quiz
3. View:
   - Number of attempts
   - Average score
   - Individual student results
4. Tap "View Details" for more information

## Troubleshooting

### Login Issues

**Problem: Can't login**
- Check your email and password
- Ensure you have internet connection
- Try "Forgot Password" if needed
- Contact your institution's IT support

**Problem: App won't load**
- Check internet connection
- Close and reopen the app
- Restart your device
- Reinstall Expo Go app

### Content Issues

**Problem: Documents won't open**
- Check internet connection
- Wait for download to complete
- Try opening again
- Report to your lecturer

**Problem: 3D models not loading**
- Ensure internet connection
- Wait a few seconds for loading
- Close and reopen PC Lab
- Try again later

### Quiz Issues

**Problem: Quiz won't submit**
- Check internet connection
- Ensure all questions are answered
- Try submitting again
- Your progress is saved automatically

**Problem: Can't see assigned quizzes**
- Refresh the dashboard
- Check if you're in the correct class
- Contact your lecturer

### Performance Issues

**Problem: App is slow**
- Close other apps
- Check available storage space
- Restart your device
- Clear app cache in Settings

**Problem: App crashes**
- Update Expo Go app
- Restart your device
- Check for app updates
- Report the issue
